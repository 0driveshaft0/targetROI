PlotMassDistribution {rawDiag}R Documentation

mass distribution plot

Description

plots the mass frequency in dependency to the charge state

Usage

PlotMassDistribution(x, method = "trellis")

Arguments

x

a data.frame fullfilling the is.rawDiag column naming criteria.

method

a character 'trellis', 'violin' or 'overlay'.

Value

a ggplot object.

Examples

 library(ggplot2)
 data(WU163763)
 
 PlotMassDistribution(WU163763, method = 'trellis') +
   facet_wrap(~ filename, ncol = 3)
 
 PlotMassDistribution(WU163763, method = 'violin') +
   theme(legend.position = 'none')
   
 PlotMassDistribution(WU163763, method = 'overlay') +
   theme(legend.position = 'none')
