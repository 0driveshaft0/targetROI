benchmark_raw {rawDiag}R Documentation

benchmark of read.raw

Description

benchmarks the read.raw function's IO. This function has been used for generating benchmark data set.

Usage

benchmark_raw(f, maxncpu = c(16, 32, 64),
  exe = file.path(path.package(package = "rawDiag"), "exec", "fgcz_raw.exe"),
  rdata = tempfile(fileext = ".RData"))

Arguments

f

set of files

maxncpu

maximal number of cores used for the benchmark

exe

the executable for reading the rawfiles

rdata

file extrention of the tempfile

Value

a nested list object.

Author(s)

Christian Panse <cp@fgcz.ethz.ch>, 2017, 2018

References

http://planetorbitrap.com/rawfilereader#.WyThDK3QPmE

Examples

## Not run: 
 f <- f[grep("raw$",f<-list.files())]
 benchmark_raw(f, maxncpu = 8, exe='~/bin/fgcz_raw.exe', mono=TRUE, rdata='/tmp/b2.RData')

## End(Not run)
