PlotMassHeatmap {rawDiag}R Documentation

mass heatmap

Description

graphs a deconvoluted heatmap of the StartTime

Usage

PlotMassHeatmap(x, method = "trellis", bins = 80)

Arguments

x

a data.frame fullfilling the is.rawDiag column naming criteria.

method

plot method.

bins

number of bins in both vertical and horizontal directions. default is 80.

Value

a gglot object.
