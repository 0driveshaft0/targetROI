PlotCycleTime {rawDiag}R Documentation

cycle time

Description

graphs cycle time versus rt. each item represents the time for one scan cycle.

Usage

PlotCycleTime(x, method = "trellis")

Arguments

x

a data.frame fullfilling the is.rawDiag column naming criteria.

method

a character 'trellis', 'violin' or 'overlay'.

Value

a ggplot object.
