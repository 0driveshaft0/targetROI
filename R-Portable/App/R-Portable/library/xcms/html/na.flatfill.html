na.flatfill {xcms}R Documentation

Fill in NA values at the extremes of a vector

Description

Extend the first and last real values in a vector to fill in any NA values present.

Usage

na.flatfill(x)

Arguments

x

numeric vector with NA values

Value

Modified vector.
